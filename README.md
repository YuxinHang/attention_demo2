# Attention Demo

Windows 上で動作する、プライバシー優先のテレビ視聴注意力デモです。原映像は推論ループのメモリ内だけで扱われ、ブラウザには顔全体をモザイク化した JPEG と匿名統計だけが送信されます。

## 起動

PowerShell で次を実行します。

```powershell
.\setup.ps1
.\start.ps1
```

ブラウザで `http://127.0.0.1:8000` を開きます。右上の「入力設定」からカメラ、ローカル動画、3秒キャリブレーションを操作できます。

## 推奨環境

- Windows 10/11
- Python 3.11
- Node.js 20 以降
- テレビ中央上部に固定した 720p USB カメラ

## プライバシー境界

- 顔認識、氏名、顔特徴量、録画、スクリーンショット、クラウド通信は実装していません。
- `/stream` は匿名化済みフレームだけを返します。
- `/ws/metrics` は人数、状態、角度、時間などの匿名数値だけを返します。
- ローカル動画はパスから直接読み込み、アップロードやコピーを行いません。
- この処理は人物の完全な匿名性や精密な眼球追跡を保証するものではありません。

## テスト

```powershell
python -m pytest
```
